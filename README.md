# 🚀 Form Submission System

## 📌 Description
Full-stack web app to perform CRUD operations on user form data using Spring Boot and PostgreSQL.

---

## 🌐 Live Links
- Frontend: https://end-to-end-form-filling-with-databa.vercel.app  
- Backend: https://end-to-end-form-filling-with-database-kne8.onrender.com/apicall/get  

---

## ⚙️ Tech Stack
- Backend: Spring Boot, Java  
- Database: PostgreSQL (Neon DB)  
- Frontend: HTML, CSS, JavaScript  
- Deployment: Render, Vercel  

---

## ✨ Features
- Add, view, update, delete data  
- REST API integration  
- Dynamic UI with Fetch API  

---

## 📡 APIs
GET /apicall/get  
POST /apicall/post  
PUT /apicall/put/{id}  
DELETE /apicall/delete/{id}
